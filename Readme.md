# Tracksy

Tracksy is a simple web application that helps you keep track of things you experience in life. You can log movies you watched, books you read, and activities you enjoyed, all in one place.

The goal of Tracksy is to create a personal history of what you consume and do, while also giving you useful insights and statistics.

---

## Features

- Add and track movies, books, and activities
- Give ratings and write short reviews
- View a timeline of your past entries
- Filter entries by type
- Personal dashboard with basic statistics


## Purpose

Tracksy was built as a personal project to practice full-stack development while creating a useful tool for tracking daily experiences.


## Tech Stack

**Frontend**
- React
- Tailwind CSS
- GSAP
- Redux
- React Query

**Backend**
- Node.js
- Express
- MongoDB
